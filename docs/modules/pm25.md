# PM2.5 (DustBoy)

แสดงค่าฝุ่น PM2.5 หน้า public — อ่านจาก **Mongo cache เท่านั้น**
DustBoy API ถูกเรียกเฉพาะใน cron

## หน้า / Components

- หน้า public/home แสดงค่าจาก cache, `components/Pmdata.js` (⚠️ root — รอเฟส 5)
- `/admin/pm25-settings` — สลับโหมดแหล่งข้อมูล (Sheet / DustBoy API)

## API / Cron

- `pages/api/pm25/*` (มี `_auth.js` ของตัวเอง)
- Cron (อย่าใส่ Clerk — ใช้ `requireCronSecret` จาก `lib/pm25CronAuth.js`):
  - `5 * * * *` รายชั่วโมง · `15 0 * * *` daily ย้อน 7 วัน · `0 3 * * *` monthly
  - เรียกจาก scheduler ภายนอกด้วย header `x-cron-secret` หรือ `?secret=`

## Models

`Pm25Latest`, `Pm25DailySnapshot`, `Pm25Monthly`, `Pm25Settings`, `Pm25SyncLog`

## Lib

`lib/pm25Sync.js` (sync logic), `lib/pm25Data.js` (อ่าน cache),
`lib/pm25CronAuth.js` (`requireCronSecret` สำหรับ cron),
`pages/api/pm25/_auth.js` — มี `requirePm25Admin` ซึ่งเป็น**ต้นแบบ auth มาตรฐาน**
ของ API ฝั่ง admin ทั้งโปรเจกต์ (getAuth → Mongo lookup → appId → allowedPages → superadmin)

## Env vars

`DUSTBOY_API_KEY`, `DUSTBOY_STATION_ID`, `DUSTBOY_API_BASE`, `CRON_SECRET`
